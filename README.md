## ğŸ¤– Pengantar Machine Learning
- **Definisi**: cabang AI yang memberi komputer kemampuan belajar dari data.  
- **Paradigma**:
  - Pemrograman tradisional â†’ aturan + data = output.  
  - ML â†’ data + output (label) = aturan.  
- **Jenis ML**:
  - Supervised Learning â†’ belajar dengan label.  
  - Unsupervised Learning â†’ menemukan pola tanpa label.  
  - Reinforcement Learning â†’ belajar dari trial & error untuk dapat **reward**.  
- **Workflow ML**:
  - Data Collection â†’ Preprocessing â†’ Model Development â†’ Evaluation â†’ Deployment â†’ Maintenance.
